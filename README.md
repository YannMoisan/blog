# Useful commands

- `bundle exec jekyll serve` Builds your site any time a source file changes and serves it locally.
- `bundle update` Updates all gems to their latest allowed versions (respecting Gemfile constraints).
- `bundle update jekyll` Update to the latest versions
- `./deploy.sh` deploy in prod

## Update ruby

- `ruby-install ruby 3.4.2`
- `chruby 3.4.2` in `~/.zshrc` chruby is purely a version switcher (no install)
- `gem install bundler` gem is the low-level installer
- `bundle install`

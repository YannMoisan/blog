A propos de ce site

Ce site a plusieurs objectifs : partager mon expérience en tant que développeur Java EE, me servir de bloc notes et mettre en pratique les nouveautés HTML 5 et CSS 3.

Ce site est volontairement simple. C'est le principe KISS. Il n'est constitué que de pages HTML statiques; autrement dit, il n'utilise ni PHP, ni MySQL. Il n'est donc pas possible de commenter les billets.

Les billets sont rédigés avec un simple éditeur de texte et les pages sont générées par un script shell maison.
